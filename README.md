# TechTrick Telegram Bot

## Features
- Verification system
- SMS sending
- Auto message system
- Admin-only group name change
- Play Hindi / Bhojpuri songs
- Buttons for YouTube, Bots, Support, Channel

## Commands
- /start → Verify subscription before using bot
- /sms number message → Send SMS
- /auto seconds message → Start auto messages
- /stop → Stop auto messages
- /autoname new_name → Change group name (Admin only)
- /bot song_name → Play Hindi/Bhojpuri song
